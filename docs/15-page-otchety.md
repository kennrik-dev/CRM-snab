# 15 · Страница «Отчёты»

Конструктор управленческих выгрузок. Опирается на агрегаты из всех блоков; расчёты — `32-calculations.md` (Этап 3). Доступ — `03-roles-permissions.md`.

---

## 1. Назначение и доступ

Выгрузки для руководства: время на этапах, суммы, просрочки, сводки по людям.

| Роль | Доступ |
|---|---|
| Руководитель, Админ | Отчёты по всем данным |
| Куратор | Отчёты — **видит данные по всем заявкам** (не ограничен своим блоком) |
| Отделы (обычные сотрудники) | **Нет доступа** к странице |

---

## 2. Макет

- Слева — **панель параметров** (тип отчёта, период, фильтры, кнопка «Сформировать»).
- Справа — **область вывода** отчёта + кнопки экспорта.

---

## 3. Параметры

- **Тип отчёта** (4 — см. §4).
- **Период:** Текущий месяц · Квартал · С начала года · Произвольный.
- **Фильтры:** тип МТР, поставщик, составитель.

---

## 4. Типы отчётов

| Тип | Содержание |
|---|---|
| Время на этапе / зависания | Заявки/процедуры с текущим этапом и числом дней на нём; флаг «зависло > N дней» |
| Суммы по этапам и поставщикам | Агрегаты сумм договоров по этапам и по поставщикам |
| Просрочки: поставки и оплаты | Просроченные поставки и УПД |
| Сводка по составителям/отделам | Кол-во заявок и суммы по людям и отделам |

---

## 5. Экспорт

**Excel · PDF · CSV.**

---

## 6. Состояния

- **Пусто:** по выбранным параметрам нет данных.
- **Нет прав:** страница недоступна обычным сотрудникам отделов.
