# 14 · Страница «Дашборд»

Сжатая сводка по всем этапам — обзорный экран. Только просмотр и переходы; редактирования нет. **Одинаков для всех ролей.** Опирается на агрегаты из всех блоков; расчётные правила — `32-calculations.md` (Этап 3).

---

## 1. Назначение и доступ

Стартовый экран единого окна: показатели в реальном времени, поток по этапам, проблемы и события, компактные таблицы. Видят **все роли одинаково**. Действий по изменению данных нет.

---

## 2. Макет (сверху вниз)

1. Полоса из **6 показателей**.
2. **Поток по этапам**.
3. Две панели: **«Требует внимания»** + **«Лента событий»**.
4. **Компактные таблицы** по этапам.

---

## 3. Показатели (6)

| Показатель | Что считает | Формат |
|---|---|---|
| В закупке | Кол-во процедур в блоке «В закупке» | число |
| В сопровождении | Кол-во процедур + сумма договоров | число + ₽ |
| Поставки в срок | Полученные в срок / всего поставок | % |
| Просрочено | Кол-во просроченных + сумма | число + ₽ |
| УПД в оплате | Кол-во УПД `Ожидает оплаты` + сумма | число + ₽ |
| УПД просрочено | Кол-во просроченных УПД + сумма | число + ₽ |

Обновляются в реальном времени.

---

## 4. Поток по этапам

Счётчики по стадиям (Ожидают закупки · В закупке · В сопровождении · Оплаты). Клик по стадии → соответствующая страница.

---

## 5. Требует внимания

Список проблем, требующих реакции: просроченные поставки, отсутствие документов, УПД без сертификата (**предупреждение — оплату провести можно**), просроченные оплаты. Клик по записи → карточка. (Точный перечень триггеров — `32-calculations.md`.)

---

## 6. Лента событий

Последние действия пользователей (кто, что, когда). Только просмотр.

---

## 7. Компактные таблицы по этапам

Сокращённые версии страниц (меньше колонок):

| Таблица | Колонки |
|---|---|
| Ожидают закупки | #, Наименование, Тип МТР, Срок, Поз., Статус |
| В закупке | #, Наименование, № заявки, Поставщик, Поз., Статус |
| В сопровождении | #, Наименование, № заявки, Поставщик, Сумма договора, Статус поставки, Просроч., Прогресс |

Клик по строке → карточка; «Открыть раздел» → полная страница. Компактной таблицы «Оплаты» нет.

---

## 8. Навигация, состояния, правила

- Дашборд **одинаков для всех ролей**.
- Только **просмотр и переходы** — редактирования на странице нет.
- Все цифры — **агрегаты в реальном времени** из всех блоков.
